# Simulação: Partícula em Campo Elétrico

![sim](https://github.com/user-attachments/assets/b7f1eb4d-23bb-4de5-9270-f57402b5efa6)

## Sobre o Projeto

Este projeto foi desenvolvido para a disciplina de Física C e consiste em uma simulação computacional construída com Processing Java. O objetivo é demonstrar visualmente a trajetória e o comportamento de uma partícula carregada ao entrar em uma região de campo elétrico uniforme. 

A simulação calcula a aceleração da partícula baseando-se na sua massa e carga, e renderiza em tempo real as alterações no seu vetor velocidade, além de deixar um rastro visual da sua trajetória.

## Controles Interativos

Durante a execução da simulação, você pode interagir com o campo elétrico usando o teclado:

* [**`V`**: Inicia a atuação do campo elétrico sobre a partícula (aplica o vetor de aceleração).
* [**`B`**: Inicia a rotação contínua do vetor do campo elétrico.

## Como Executar

1. Certifique-se de ter o [Processing](https://processing.org/download) instalado.
2. Clone este repositório.
3. Abra o arquivo principal `FisicaC_simulacao.pde` na IDE do Processing.
4. [cite_start]Clique em **Run** para iniciar a simulação.
